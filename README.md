## What It Does

Our Space lets users explore the solar system interactively — browsing planets, learning about moons, and discovering stars through rich visual content powered by NASA's public APIs. Designed to make space education engaging on mobile.

### Features
- 🪐 Browse all planets in the solar system with detailed information
- 🌙 Explore moons and their relationships to parent planets
- ⭐ Discover stars and their classifications
- 📡 Live data from NASA's public APIs
- 🎨 Immersive space-themed UI with smooth animations
